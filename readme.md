		> 红黑树是每个节点都带有颜色属性的二叉查找树，颜色或红色或黑色,在二叉查找树强制一般要求以外，对于任何有效的红黑树我们增加了如下的额外要求:  
			1. 节点是红色或黑色  
			2. 根节点是黑色  
			3. 所有叶子都是黑色(叶子是NUIL节点)  
			4. 每个红色节点的两个子节点都是黑色(从每个叶子到根的所有路径上不能有两个连续的红色节点)  
			5. 从任一节点到其每个叶子的所有路径都包含相同数目的黑色节点
